用户交友网站

介绍：帮助大家找到志同道合的伙伴，移动端H5

需求分析

  1. 用户去添加标签，标签的分类（要有哪些标签，怎么把标签进行分类）
  2. 主动搜索：允许用户根据标签搜索其他用户
    1. Redis缓存 + 本地
  3. 组队：
    1. 创建队伍
    2. 加入队伍
    3. 根据标签查询队伍
    4. 邀请其他人
  4. 推荐
    1. 相似度计算算法 + 本地分布式计算
  5. 允许用户修改标签

技术栈

前端

  1. Vue3开发框架（提高页面开发效率）
  2. Vant UI（基于Vue的移动组件库）（React版）
  3. Vite（打包工具，快）
  4. Nginx来单机部署

后端

  1. Java编程语言+SpringBoot框架
  2. SpringMVC + MyBatis + Mybatis Plus
  3. MySQL 数据库
  4. Redis缓存
  5. Swagger + Knife4j接口文档

第一期计划

  1. 前端项目初始化
  2. 数据库表设计
  3. 前端主页
  4. 数据库表设计
    1. 标签表
    2. 用户表
  5. 开发后端 - 根据标签搜索用户
  6. 开发前端 - 根据标签搜索用户
