line-height

line-height（行高）是指文本行与行之间的垂直空间，通常用于排版和网页设计中。它也被称为传统排版中的“行间距”（leading）。通过调整line-height，可以改变文本行之间的垂直间距，从而影响文本的整体外观和可读性。

vh是什么

vh 是 CSS 中的长度单位，表示相对视口高度的百分比。一个视口（viewport）是浏览器中用于显示网页的区域，而 vh 单位表示视口的高度的百分比。

具体来说，1 vh 等于视口高度的 1%，100 vh 等于整个视口的高度。这个单位通常用于创建基于视口高度的动态和响应式布局。例如，如果你想确保一个元素的高度始终是视口高度的一半，你可以这样设置：

display

在 CSS 中，display 属性用于定义 HTML 元素的显示方式。这个属性决定了元素是块级元素还是内联元素，或者是一些其他特殊的显示方式。

以下是一些常见的 display 属性值：

  1. block： 元素呈块级显示，即它会在页面上生成一个块，宽度默认是父元素的宽度，可以设置宽度和高度。
  1. inline： 元素呈内联显示，即元素会在同一行内显示，不会独占一行，宽度和高度只取决于内容。
  1. inline-block： 元素呈内联块级显示，它融合了内联元素和块级元素的特性，可以设置宽度和高度，同时在同一行内显示。
  1. none： 元素不会被显示，即元素在页面中不可见，也不占据空间。
  1. flex： 使用 Flexbox 布局模型，使得元素能够更方便地布局和对齐子元素。
  1. grid： 使用 Grid 布局模型，提供了更复杂的布局能力，可以在二维空间中定义和对齐元素。

transform

在 CSS 中，transform 属性用于对元素进行变换，包括平移、旋转、缩放和倾斜。这个属性允许你以更复杂的方式改变元素的外观，而无需改变其文档流布局。

以下是一些常见的 transform 属性的值和用法：

  1. 平移 (Translate)： translate 函数用于沿着 X 和 Y 轴平移元素。
  1. 旋转 (Rotate)： rotate 函数用于旋转元素。角度可以是正数（顺时针）或负数（逆时针）。
  1. 缩放 (Scale)： scale 函数用于缩放元素。可以分别设置水平和垂直方向的缩放比例。
  1. 倾斜 (Skew)： skew 函数用于倾斜元素。可以分别设置水平和垂直方向的倾斜角度。
  1. 变换原点 (Transform Origin)： transform-origin 属性用于指定变换的原点，默认是元素的中心点。
  1. 多重变换 (Multiple Transforms)： 可以组合多个变换，用空格分隔。

这些变换可以单独或组合使用，让你能够实现各种独特的元素动画和布局效果。值得注意的是，transform 属性是一个累加的属性，多个变换会按照顺序依次应用。
